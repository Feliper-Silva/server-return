## :rocket: Como executar

```sh
# Clone o repositório
git clone https://github.com/Feliper-Silva/server-return.git
```

```sh
# baixar dependências
npm install
```

Crie as chaves no arquivo .env

```sh
# iniciar a aplicação.
npm run dev
```

Por fim, a aplicação estará disponível em http://localhost:3333
